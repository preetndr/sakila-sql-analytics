# Sakila SQL Analytics

A structured collection of SQL queries performed on the Sakila sample database using MySQL Workbench.  
This project focuses on core relational database concepts including joins, aggregations, filtering, subqueries, and database manipulation commands.

---

## Project Overview

This repository contains beginner-to-intermediate SQL exercises designed to explore relational database operations using the Sakila database.

The queries demonstrate practical usage of:
- Multi-table joins
- Aggregate functions
- Grouping and filtering
- Subqueries
- Conditional logic
- Data definition commands

---

## Technologies Used

- MySQL
- MySQL Workbench
- Sakila Sample Database
- Git & GitHub

---

## SQL Concepts Covered

| Concept | Description |
|---|---|
| SELECT | Retrieve data from tables |
| JOIN | Combine related tables |
| LEFT JOIN | Include unmatched rows |
| WHERE | Filter records |
| GROUP BY | Group related rows |
| HAVING | Filter grouped results |
| COUNT / SUM | Aggregate calculations |
| Subqueries | Nested query execution |
| CASE Statements | Conditional logic |
| LIKE | Pattern matching |
| CREATE / ALTER / DROP | Database structure operations |
